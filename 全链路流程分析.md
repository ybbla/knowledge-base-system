### 2.5 POST /documents/{id}/restore — 恢复文档

```
restore_document(doc_id)
│

│   └── 不存在 → 404
│
├── 读取 previous_status（由 soft_delete 写入 metadata）
```


### 4.2 递进切分：_split_recursive()

```

│
├── [前置] token < safe_threshold?
│   └── YES → 尝试 _extract_section() → LLM
│       └── 成功且有内容 → return chunks（当前段已足够小，不再下钻）
│
├── [切分] 在 level 层级标题处切分 → sections
```
